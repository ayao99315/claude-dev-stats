## 🏃‍♂️ 快速运行示例

```bash
# 克隆项目并安装依赖
git clone https://github.com/your-repo/claude-dev-stats.git
cd claude-dev-stats
npm install

# 编译示例（TypeScript）
npm run build

# 运行基础示例
node dist/examples/basic-usage.js

# 或直接使用 ts-node 运行
npx ts-node examples/basic-usage.ts
```
